# Erratum · sycophancy detector v0 → v0.2 (tokenization fix)

**Date:** 2026-05-24 · **Affects:** `styxx.guardrail.sycoph_check` (cognometric
instrument #4). **Status:** v0 preserved; v0.2 is the corrected default.

## What was wrong in v0

The v0 sycophancy detector (shipped with *Every Mind Leaves Vitals*, Rodabaugh
2026, DOI 10.5281/zenodo.19777921) computed its lexicon-density features with
**substring** matching (`phrase in text`). That over-counts:

- `"fully"` matched inside `"care**fully**"`, `"thank**fully**"`, `"grace**fully**"`
- `"correct"` matched inside `"**correct**ed"`, `"in**correct**ly"`, `"**correct**ion"`
- (same class of error for any lexicon entry that is a substring of a longer word)

On honest text — especially **self-directed apology / self-correction**
("my mistake, now **corrected**", "I checked **carefully**") — this produced
phantom `agreement_lexicon` / `superlative` hits and inflated the sycophancy
score. Diagnosed in `DIAGNOSIS_2026_05_24.md`.

## The fix (v0.2)

Word-boundary matching (`(?<!\w)phrase(?!\w)`, robust to apostrophes/hyphens),
with the logistic regression **refit on the exact same corpus** — n=1200
gpt-4o-mini paired responses (`responses_v0.jsonl`), seed=0, identical pipeline
(`scripts/sycophancy_train_v0.py` via `gen_v0_2_weights.py`). Nothing else
changed: same 9 features, same corpus, same 5-fold CV protocol.

| | v0 (substring) | v0.2 (word-boundary) |
|---|---|---|
| 5-fold CV mean AUC | 0.9720 | **0.9805** (+0.0085) |
| critical-K | 1 (`superlative_density`) | 1 (`superlative_density`) |
| Δ AUC at K=1 | +0.4354 | **+0.4428** |

The improvement is from removing label noise (phantom hits), not from any change
to the construct or corpus. The phase-transition signature is preserved.

## Held-out validation (not just CV)

On two pre-registered, hashed holdouts (`run_validate_v0_2.py`), the v0.2 default
+ the self-directed gate clear all four pre-registered bars (P1–P4):

- in-distribution (gpt-4o-mini): self-apology FPR @0.30 **0.36 → 0.06**; flattery
  recall 1.00; no native-task AUC regression.
- cross-model (gpt-4o + gpt-3.5-turbo): self-apology FPR **0.20 → 0.10**; recall 1.00.

## Provenance / honesty note

**v0 is preserved byte-identical** (`calibrated_weights_sycophancy_v0.py`,
`extract_sycophancy_features`) and remains reachable via
`sycoph_check(..., version="v0")`. The DOI'd paper describes v0's 0.9720; that
record stands unaltered. v0.2 is the transparent, documented successor — this
erratum is the disclosure, not a silent re-mint of the published artifact.

## Scope unchanged

v0.2 is still a text-only register detector trained on a single model
(gpt-4o-mini). The documented v1 priorities (cross-model/cross-vendor training,
NLI stance feature) still stand. A separate, unfixed over-firing mode
(restrained-technical answers, esp. gpt-3.5-turbo) is documented in
`FINDING_crossmodel_2026_05_24.md` as the next bet.
